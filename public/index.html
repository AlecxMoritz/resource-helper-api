Resource Helper Endpoints

User Controller

Topics Controller

get all by user_uid -- https://resource-helper-api.herokuapp.com/topic/

get one by user_uid -- https://resource-helper-api.herokuapp.com/topic/:uid

new topic -- https://resource-helper-api.herokuapp.com/topic/new

update topic -- https://resource-helper-api.herokuapp.com/topic/update/:uid

delete topic -- https://resource-helper-api.herokuapp.com/topic/delete/:uid

Resource Controller

get all by user_uid -- https://resource-helper-api.herokuapp.com/resource/

get all by topic_uid -- https://resource-helper-api.herokuapp.com/resource/topics/u:id

new resource -- https://resource-helper-api.herokuapp.com/resource/new

update resource -- https://resource-helper-api.herokuapp.com/resource/update/:uid

delete resource -- https://resource-helper-api.herokuapp.com/resource/delete/:uid
