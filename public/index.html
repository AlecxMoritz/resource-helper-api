Resource Helper Endpoints

User Controller

sign up -- http://localhost:4200/user/signup

sign in -- http://localhost:4200/user/signin

Topics Controller

get all by user_uid -- http://localhost:4200/topic/

get one by user_uid -- http://localhost:4200/topic/:uid

new topic -- http://localhost:4200/topic/new

update topic -- http://localhost:4200/topic/update/:uid

delete topic -- http://localhost:4200/topic/delete/:uid

Resource Controller

get all by user_uid -- http://localhost:4200/resource/

get one by uid http://localhost:4200/resource/:uid

get all by topic_uid -- http://localhost:4200/resource/topics/u:id

new resource -- http://localhost:4200/resource/new

update resource -- http://localhost:4200/resource/update/:uid

delete resource -- http://localhost:4200/resource/delete/:uid
